# Kotlin Kurs Proje Ödevi (Android Studio Kotlin İlk Projem)

+ Atıl Samancıoğlu'ndan aldığım "Android Mobil Uygulama Kursu: Kotlin & Java" kursundaki ikinci ödevimi paylaşıyorum.
